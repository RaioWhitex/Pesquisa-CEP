# 📝 Formulário de Cadastro

Um formulário de cadastro de endereço que preenche automaticamente os campos a partir do CEP informado, com interface limpa, minimalista e responsiva.

Ideal para cadastros, checkouts e qualquer aplicação que precise capturar endereços de forma rápida.

## ✨ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## 🎨 Características do Design

- Visual minimalista e clean
- Campos com linha de base discreta
- Rótulos pequenos acima de cada campo
- Tipografia leve e de fácil leitura
- Layout organizado e responsivo

## 🚀 Objetivo do Projeto

Praticar a construção de formulários interativos, explorando consumo de API de CEP (ViaCEP), preenchimento automático de campos e estilização com CSS.

## 📚 Funcionalidades

- Campo de CEP com preenchimento automático
- Preenchimento de rua, bairro, cidade e estado
- Campo manual para número do endereço
- Validação dos dados informados
- Interface leve e responsiva

## 💼 Possíveis Aplicações

- Cadastro de clientes e usuários
- Formulário de endereço em checkouts
- Componente para sistemas de entrega
- Cadastro em landing pages
- Ferramenta de consulta de CEP
